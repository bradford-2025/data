This is the project documentation for the
[Bradford 2025](https://bradford2025.co.uk) data pipelines.

It describes the way data is taken from the operational systems, processed and
published as open data files.

This site contains the 

1. [Overview](overview.md)  
   An overview of the processing, focussing on the source systems.
2. [Data](datasets/overview.md)  
   Information about the extracted from the system. This will also act as the
   publication site for the datasets as they are openly licensed.
3. [Pipelines](pipelines.md)  
   Documentation of individual pipelines, and overview of the orchestration.
4. [Reference](reference.md)  
   Reference to the utility code the pipelines depend on.
5. [How-To Guides](how-to-guides.md)  
   Guides to help use and support the pipeline code.
6. [Architecture](architecture/overview.md)  
   Overview of the architecture of the data extract and publication.
