This is the project documentation for the
[Bradford 2025](https://bradford2025.co.uk) data pipelines.

It describes the way data is taken from the operational systems, processed and
published as open data files.

NB at the moment this is work in progress, so is incomplete in some areas

Quickly find what you're looking for depending on your use case by looking at
the different pages.

1. [Overview](overview.md)  
   An overview of the processing, covering source systems, pipeline stages and
   outputs.
2. [Pipelines](pipelines.md)  
   Documentation of individual pipelines.
3. [Reference](reference.md)  
   Reference to the utility code the pipelines depend on.
4. [How-To Guides](how-to-guides.md)  
   Guides to help use and support the pipeline code.
